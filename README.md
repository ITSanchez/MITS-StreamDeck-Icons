# 🎨 MITS StreamDeck Icons

Free and Open Source icon pack for **Stream Deck**, **Redragon SS552**, **Touch Portal**, **Loupedeck**, and other programmable keypads that support custom PNG icons.

---

## 🇺🇸 English

### ✨ Features

- 🎬 DaVinci Resolve icons
- 🎹 Logic Pro icons
- 🎵 n-Track Studio icons
- 🪟 Windows icons
- 🍎 macOS icons
- ⚙️ General-purpose icons

### 📦 Included Variants

- ⚪ Mono
- 🎨 Accent
- 🌈 Color

### 📐 Available Sizes

- 72 × 72 px
- 96 × 96 px
- 144 × 144 px

### ✅ Compatible With

- Elgato Stream Deck
- Redragon SS552
- Touch Portal
- Loupedeck
- Any programmable keypad supporting PNG icons
